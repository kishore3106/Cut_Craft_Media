## 🎬 About

**Cut Craft Media** is a boutique video editing and post-production studio. This portfolio website is built to showcase my freelance editing work across a wide range of categories — from high-energy social media reels to cinematic brand narratives.

The site features a sleek dark aesthetic, smooth scroll animations, a 3D video carousel, and interactive project cards — all designed to let the work speak for itself.

## 🛠️ Tech Stack

- **React 19** + **TypeScript**
- **Vite** (build tool)
- **Tailwind CSS v4**
- **Framer Motion / motion-react** — scroll animations, 3D carousel, parallax effects
- **Lenis** — buttery smooth scrolling
- **React Router DOM** — page navigation
- **Lucide React** — icons

---

## 🚀 Run Locally

**Prerequisites:** Node.js

```bash
# Install dependencies
npm install

# Start the dev server
npm run dev
```

Visit `http://localhost:3000`
